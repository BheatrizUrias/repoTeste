Passo a Passo para Colaboração com Git

1. Configurar Acessos

Peça acesso ao repositório e configure sua chave SSH ou token de autenticação.

2. Clonar o Repositório

Baixe o código para sua máquina:

git clone <URL_DO_REPO>

3. Criar uma Nova Branch

Trabalhe em uma branch separada para cada tarefa:

git checkout -b minha-feature

4. Fazer Alterações e Commitar

Edite os arquivos, adicione e faça commit:

git add .
git commit -m "Descrição da mudança"

5. Sincronizar com o Repositório Remoto

Antes de enviar, atualize sua branch:

git pull origin main --rebase

6. Enviar as Alterações

Suba sua branch para o repositório remoto:

git push origin minha-feature

7. Criar um Pull Request (PR)

No GitHub/GitLab, crie um PR da sua branch para a branch principal.

8. Revisão e Ajustes

Receba feedback, corrija se necessário e atualize o PR.

9. Merge na Branch Principal

Após aprovação, faça merge da branch:

git checkout main
git pull origin main
git merge minha-feature
git push origin main

10. Baixar Atualizações

Antes de começar uma nova tarefa, sempre atualize o repositório:

git checkout main
git pull origin main
